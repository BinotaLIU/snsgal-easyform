# SNSGAL

![Build Status](https://travis-ci.com/BinotaLIU/snsgal-easyform.svg?token=pyTLBzEbQYxLQwzeYMoH&branch=master)

## Building
```
composer install
php artisan key:generate
cp .env.example .env
vi .env
php artisan migrate
yarn install
gulp --production # or just gulp if you're in developing
```
